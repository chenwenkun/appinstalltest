Mac 本地环境配置指南

1. 环境准备

Python 环境: 建议安装 Python 3.8+

python3 --version

2. Android 环境 (ADB)

推荐使用 Homebrew 安装（若已安装则跳过）:

brew install android-platform-tools

连接手机: 开启"开发者模式" -> 开启"USB 调试" -> 授权电脑。

验证:

adb devices

应显示 device 状态

3. 安装与运行

安装最新版工具:

pip install appinstalltest --upgrade -i https://pypi.doubanio.com/simple

启动服务:

appinstalltest

服务启动后会自动跳转到控制台。
