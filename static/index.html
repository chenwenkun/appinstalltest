uiauto.dev

uiauto.dev帮助你快速编写App UI自动化脚本，它由网页端+客户端组成

  • 支持Android和iOS
  • 支持鼠标选择控件查看属性，控件树辅助精确定位，生成选择器
  • 支持找色功能，方向键微调坐标，获取RGB、HSB

QQ群: 536481989

安装

1. 安装Python 3.8+

2. 安装并启动

pip3 install -U uiautodev -i https://pypi.doubanio.com/simple

                # or: python3 -m uiautodev

uiauto.dev正常启动后，刷新网页即可访问本地页面。
